Security & Privacy Policy

Purpose of This Page

This Security & Privacy Policy explains how this website operates and ensures a safe, transparent, and secure experience for all users.

Zero Data Collection No personal information is ever collected, stored, or shared
Static QR Codes Permanent links with no redirect risks or tracking parameters
Secure Hosting GitHub Pages with HTTPS encryption and DDoS protection
Client-Side Only No server scripts, databases, or backend processing

Data Collection

  • This website does NOT collect any personal data whatsoever.
  • No login, signup, cookies, analytics, or tracking systems are used.
  • No names, phone numbers, email addresses, locations, or device info are stored.
  • All interactions happen locally in your browser with no external transmission.

QR Code Safety

All QR codes used in this project are static and permanently linked to GitHub Pages URLs. They:

  • Do not redirect to unknown, suspicious, or third-party websites
  • Contain only educational, non-commercial information about campus trees
  • Are safe for students, teachers, parents, and all visitors to scan
  • Cannot execute code, download files, or access your device

Website Hosting & Security

This website is hosted on GitHub Pages, a secure and trusted platform by Microsoft.

  • HTTPS encryption is enabled for all connections
  • No server-side scripts (PHP, Python, Node.js) are used
  • No advertisements, pop-ups, or monetization scripts present
  • Regular security audits and malware scanning by GitHub
  • Globally distributed CDN for fast, reliable access

Content Transparency

What content is shown on this website:

  • Educational information about trees (names, scientific classification, benefits)
  • Environmental awareness messages aligned with NSS values
  • Official school and NSS branding for identification purposes
  • Project credits, policies, and contact information
  • NO personal data, tracking pixels, or hidden content

All content is publicly viewable, non-sensitive, and education-focused.

Content Usage

  • All content is for educational and awareness purposes only.
  • The information provided is strictly non-commercial and non-profit.
  • Images and logos are used with permission for educational representation only.
  • Reproduction for classroom teaching is permitted with proper attribution.

Responsible Use

Users are encouraged to use this website responsibly and respectfully. Any misuse, tampering, or misrepresentation of content is discouraged.

  • Do not attempt to modify, scrape, or automate access to this website
  • Do not use content for commercial purposes without explicit permission
  • Report any security concerns to the project maintainers immediately
  • Respect the educational intent and community-focused nature of this project
