Security & Privacy Policy

NSS Tree QR Awareness Project

Purpose of This Page

This Security & Privacy Policy explains how this website operates and ensures a safe, transparent, and secure experience for all users.

Data Collection

  • This website does NOT collect any personal data.
  • No login, signup, cookies, or tracking systems are used.
  • No names, phone numbers, email addresses, or locations are stored.

QR Code Safety

All QR codes used in this project are static and permanently linked to GitHub Pages URLs. They:

  • Do not redirect to unknown or third-party websites
  • Contain only educational information
  • Are safe for students, teachers, and visitors

Website Hosting & Security

This website is hosted on GitHub Pages, a secure and trusted platform.

  • HTTPS encryption is enabled
  • No server-side scripts are used
  • No malware, ads, or pop-ups are present

Content Usage

  • All content is for educational and awareness purposes only.
  • The information provided is non-commercial.
  • Images and logos are used strictly for educational representation.

Responsible Use

Users are encouraged to use this website responsibly and respectfully. Any misuse, tampering, or misrepresentation of content is discouraged.
