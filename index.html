üìå Project Overview

The School Trees QR Code Project is an eco-friendly and educational initiative aimed at spreading awareness about trees present inside the school campus.

Each tree is tagged with a QR code.
When scanned, the QR code opens a dedicated web page that displays:
  • Tree name
  • Scientific name
  • Features and benefits
  • Environmental importance
  • Awareness messages
This initiative combines technology with environmental responsibility.
